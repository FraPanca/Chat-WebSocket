# Chat-WebSocket

## Italiano  
Sviluppo di un servizio di chat di gruppo web basato su WebSocket.

### Descrizione  
È possibile accedere a una chat di gruppo inserendo il proprio username e l'identificatore numerico del gruppo.  

Se il gruppo non esiste, viene automaticamente creato.  

La comunicazione avviene in tempo reale grazie all'utilizzo di WebSocket.

---

## English  
Development of a web-based group chat service built using WebSocket.

### Description  
Users can join a group chat by providing a username and a numeric group identifier.  

If the specified group does not exist, it is automatically created.  

Communication occurs in real time through WebSocket technology.
